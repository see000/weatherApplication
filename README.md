# Weather Application

A responsive weather app for Android and iOS devices.

## Features

- **Weekly Forecasts:** Get detailed weather information for the next 7 days.
- **Offline Access:** Store data locally for offline use, ensuring access even without an internet connection.
- **Dynamic UI:** Background images adapt to predicted weather conditions, creating a visually immersive experience.
- **Location Awareness:** Automatically detects your location or allows manual search for specific cities.
- **User-Friendly Interface:** Simple and intuitive design makes it easy to find the information you need.

## Technologies

- Flutter
- Bloc (State Management)
- Hive (Local Storage)
- Geolocator (Location Services)
- Various UI/UX libraries

## Key Accomplishments

- Built a fully responsive app that adapts to different screen sizes and devices.
- Integrated with weather APIs to provide accurate forecasts.
- Implemented local data storage for offline accessibility.
- Designed a user-friendly interface with dynamic background images that enhance the user experience.
